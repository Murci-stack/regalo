# Tu regalo para Fina

Una pequeña web familiar para celebrar el 77 cumpleaños de Fina y entregarle su regalo de una forma especial.

## Contenido

- Presentación optimizada para móvil.
- Dedicatoria familiar.
- Galería de quienes entregan el regalo.
- Animación y confeti respetando `prefers-reduced-motion`.
- Acceso directo al documento `REGALO.pdf`.

## Vista local

No requiere instalación ni compilación. Sirve esta carpeta con cualquier servidor estático y abre `index.html`.

## Privacidad

La página solicita a los buscadores que no la indexen mediante `robots.txt`, metaetiquetas y la cabecera `X-Robots-Tag`. Aun así, cualquier persona que conozca el enlace público puede abrirla.
